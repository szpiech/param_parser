CommandLineArgumentParser.hpp is a header file that implements a simple 
command line argument parser.

Note, all functionality and methods are unit tested in 
unit_test_command_line_argument_parser.cpp

To compile and test:

g++ -o unit_test_command_line_argument_parser unit_test_command_line_argument_parser.cpp CommandLineArgumentParser.hpp
./unit_test_command_line_argument_parser
